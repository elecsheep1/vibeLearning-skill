# 英语

## 元信息
- learning_mode: exam
- description: 考研英语，涵盖词汇、语法、阅读、写作、翻译等。

## 常见错误类型分类

### 词汇
- **近义词混淆**：如 affect/effect, principal/principle, complement/compliment。应从词根+语境区分。
- **固定搭配错误**：如 depend on 写成 depend of，congratulate on 写成 congratulate for。
- **词性误用**：名词当动词用，形容词当副词用（如 "He runs quick" → "He runs quickly"）。

### 语法
- **时态混用**：主从句时态不一致。典型：主句过去时，从句却用现在时。
- **虚拟语气遗漏**：if 条件句中该用 were/had done 却用了 was/did。
- **主谓一致错误**：主语是单数但谓语用复数，尤其当主语后跟介词短语时（"The list of items are..." → "is"）。
- **非谓语动词混淆**：
  - doing（主动/进行）
  - done（被动/完成）
  - to do（目的/将来）
- **定语从句关系词误用**：that vs which（限定性/非限定性），who vs whom。

### 阅读
- **过度推断**：选了一个原文没有直接支持、需要额外假设的选项。
- **以偏概全**：用局部细节替代全文主旨。
- **定位错误**：在错误段落找答案，没有根据题干关键词准确定位。

### 写作
- **Chinglish 句式**：按中文语序直译，如 "I very like it" → "I like it very much"。
- **连词缺失/滥用**：流水句（缺少连接词）或 however 当连词用。
- **模板生搬硬套**：套用模板但与题目要求不匹配。

### 翻译
- **拘泥于字面**：逐词翻译导致不通顺。应先理解语义，再用目标语言自然表达。
- **长难句拆解失败**：遇到嵌套从句时无法理清主干。

## 诊断规则

- 学生作文中出现大量短句 → 缺乏从句连接能力，需练习定语从句和状语从句
- 阅读理解正确率低但词汇量尚可 → 问题可能在长难句解析，需训练找主干
- 翻译时语序怪异 → 英汉语序差异意识不足（英语前重心，汉语后重心）
- 完形填空错误集中在虚词 → 语法基础薄弱，尤其是介词和连词

## 推荐解题步骤模板

### 阅读理解
1. 先读题干（不含选项），圈出关键词
2. 带着问题扫读文章，定位关键词所在段落
3. 精读定位段落，理解句间逻辑
4. 排除法消去 3 个错误选项（常见错误：无中生有、张冠李戴、过度推断、偷换概念）
5. 确认最佳答案

### 写作（短文）
1. 审题：确定文体、时态、人称、要点
2. 列提纲：开头（2 句）- 主体（要点展开）- 结尾（1-2 句）
3. 写初稿：先求准确再求高级
4. 检查和升级：时态一致性、主谓一致、连接词、词汇替换
5. 誊写

### 长难句分析
1. 找谓语动词（时态/语态标记）
2. 找连词（并列连词 / 从属连词）→ 划分从句边界
3. 提取主干：主语 + 谓语 + 宾语/表语
4. 逐层添加修饰成分理解

## 知识点关联图

```
词汇（词根词缀、词性、搭配）
    ↓
语法（词法 → 句法）
    ├── 时态语态
    ├── 从句（名/形/副）
    ├── 非谓语
    └── 虚拟语气
    ↓
技能应用
    ├── 阅读（主旨、细节、推断、猜词）
    ├── 写作（审题、结构、句式、词汇）
    ├── 翻译（理解 → 表达 → 润色）
    └── 听力（辨音 → 预测 → 速记）
```

- **词汇**是砖，**语法**是框架
- 阅读和听力是**输入**技能，依赖词汇+长难句解析
- 写作和翻译是**输出**技能，依赖词汇+语法+思维转换
- 输入技能提升可带动输出技能

## 重点/难点/常考点

- **重点**：核心词汇（3500/5500）、基本语法体系、阅读定位能力
- **难点**：长难句分析、虚拟语气、非谓语动词在写作中的运用
- **常考点**：阅读理解（细节题+主旨题）、完形填空（上下文逻辑）、写作（书信+议论文）
