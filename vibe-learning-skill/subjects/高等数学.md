# 高等数学

## 元信息
- learning_mode: exam
- description: 考研数学一，涵盖微积分、级数、常微分方程等。

## 常见错误类型分类

### 极限与连续
- **极限符号遗漏**：计算极限时忘记写 `lim` 符号，直接写表达式。纠正：每一步保持 `lim` 直到代入。
- **等价无穷小滥用**：在加减运算中使用等价无穷小替换（只能在乘除中使用）。典型表现：`sin(x) - x` 直接用 `x - x = 0` 替换。
- **极限存在性未验证**：直接使用极限四则运算法则，但未先验证各部分极限是否存在。

### 导数与微分
- **链式法则遗漏**：复合函数求导时忘记乘以内层函数的导数。典型表现：求 `sin(2x)` 的导数只写 `cos(2x)`。
- **隐函数求导漏项**：对 y 求导时忘记 `y` 本身是 `x` 的函数，缺少 `dy/dx` 因子。
- **高阶导数符号混淆**：`f''(x)` vs `d²y/dx²` 的记法混淆。

### 积分
- **不定积分遗漏常数 C**：计算不定积分后忘记加积分常数。
- **定积分换元未换限**：使用换元法时忘记同步更新积分上下限。
- **分部积分选 u 不当**：选取的 u 和 dv 导致积分越来越复杂。应按"反对幂指三"顺序选 u。

### 微分方程
- **特征方程写错**：二阶常系数齐次方程的特征方程系数符号写错。
- **特解形式设错**：非齐次项与齐次解重复时，忘记乘 x 升阶。

### 级数
- **收敛域端点漏判**：求幂级数收敛域时只求收敛半径，忘记单独判断端点处的敛散性。

## 诊断规则

- 用户解题结果差一个常数 → 大概率漏了积分常数 C
- 用户求导结果比标准答案少一个因子 → 链式法则遗漏
- 用户在极限计算中得到 0/0 但无法继续 → 可能未掌握洛必达法则或等价无穷小
- 用户积分上下限代入后结果为负但面积应为正 → 可能被积函数在区间内变号

## 推荐解题步骤模板

### 极限计算
1. 判断极限类型：确定型 / 未定型（0/0, ∞/∞, 0·∞, ∞-∞, 1^∞, 0^0, ∞^0）
2. 若是未定型，选择方法：等价无穷小 / 洛必达 / 重要极限 / 夹逼定理
3. 逐步化简，保持极限符号
4. 得到结果后验证合理性

### 不定积分
1. 观察被积函数结构
2. 判断方法：直接积分 / 换元 / 分部 / 有理函数分解 / 三角代换
3. 执行计算
4. **检查：加 C**

### 定积分应用（求面积/体积）
1. 画图（或想象）区域形状
2. 确定积分变量和上下限
3. 列出微元（面积微元 dA 或体积微元 dV）
4. 定积分计算
5. 验证结果非负且量级合理

## 知识点关联图

```
极限 → 连续 → 导数 → 微分 → 不定积分 → 定积分 → 广义积分
                                    ↓
                              微分方程 ← 定积分应用（面积/体积/弧长）
                                   
级数 → 幂级数 → 傅里叶级数
  ↓
泰勒展开 → 近似计算

多元函数 → 偏导数 → 全微分 → 二重积分 → 三重积分 → 曲线曲面积分
```

- **极限** 是所有分析的基石
- **导数** 依赖极限定义
- **不定积分** 是导数的逆运算
- **定积分** 由原函数（不定积分）+ Newton-Leibniz 公式计算
- **常微分方程** 本质上是不定积分的延伸
- **级数** 是离散求和到连续积分的桥梁

## 重点/难点/常考点

- **重点**：极限计算、导数应用（单调性/极值/最值）、不定积分与定积分计算
- **难点**：中值定理证明题、级数敛散性判别、曲线曲面积分
- **常考点**：洛必达法则、Taylor 展开求极限、定积分应用（旋转体体积）、一阶线性微分方程
