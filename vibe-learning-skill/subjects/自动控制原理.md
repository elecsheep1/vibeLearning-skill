# 自动控制原理

## 元信息
- learning_mode: exam

## 常见错误类型分类

### 数学模型
- **传递函数推导遗漏初始条件**：从微分方程推导传递函数时，忘记假设零初始条件。
- **梅森增益公式符号错误**：计算信号流图时，回路增益正负号混淆。规则：负反馈回路增益取负。

### 时域分析
- **劳斯判据符号错误**：劳斯表第一列变号次数判断错误。典型表现：第一列元素符号改变 n 次 → 有 n 个右半平面极点。
- **劳斯表全零行处理错误**：遇到全零行时忘记用辅助方程求导后的系数替代。
- **稳态误差系数混淆**：Kp（位置误差常数）、Kv（速度误差常数）、Ka（加速度误差常数）分别对应阶跃、斜坡、抛物线输入。
- **二阶系统超调量公式记反**：`σ% = e^(-πζ/√(1-ζ²))`，其中 ζ 越小超调越大。

### 根轨迹
- **根轨迹分支数判断错误**：分支数 = max(开环极点数, 开环零点数)。
- **渐近线与实轴交点公式错误**：`σa = (Σ极点 - Σ零点) / (n - m)`，容易分子分母写反。
- **分离点/会合点遗漏**：只求出了实轴上的分离点，忽略了共轭复数的分离点。

### 频域分析
- **Nyquist 曲线包围圈数数错**：N = Z - P，其中 N 是逆时针包围 (-1, j0) 的圈数。
- **Bode 图转折频率计算错误**：一阶因子的转折频率是 ω = 1/T，不是 ω = T。
- **稳定裕度读图不准确**：在 Bode 图上，相位裕度是 ωc 处相位离 -180° 的距离。

### 校正与综合
- **超前/滞后校正选择不当**：
  - 超前校正 → 改善动态性能（提高相位裕度）
  - 滞后校正 → 改善稳态精度（提高低频增益）
- **PID 参数整定顺序错误**：应先调 P，再 D，最后 I。先加 I 容易引起振荡。

## 诊断规则

- 学生劳斯表总是算不对 → 检查是否在首列处理上出错，尤其是符号
- 根轨迹画不出来或画不全 → 先确认开环极点和零点是否全部找出
- Nyquist 图判断稳定性出错 → 大概率是圈数方向搞反了
- 相位裕度算出来和仿真对不上 → 可能混淆了 ωc（穿越频率）和 ωg（相位交界频率）

## 推荐解题步骤模板

### 时域分析（二阶系统）
1. 写出闭环传递函数 Φ(s)
2. 与标准二阶形式对比：`Φ(s) = ωn² / (s² + 2ζωn·s + ωn²)`
3. 求出 ζ 和 ωn
4. 根据公式计算：上升时间 tr、峰值时间 tp、超调量 σ%、调节时间 ts
5. 判断系统类型（欠阻尼/临界阻尼/过阻尼）

### 画根轨迹
1. 标出开环极点和零点（× 和 ○）
2. 确定实轴上的根轨迹段（右方极零点总数为奇数）
3. 计算渐近线（交点 + 角度）
4. 求分离点/会合点
5. 求与虚轴的交点（用劳斯判据或令 s=jω）
6. 画出根轨迹草图

### 频域稳定性分析
1. 写出开环频率特性 G(jω)H(jω)
2. 画出 Nyquist 曲线（或 Bode 图）
3. 确定 (-1, j0) 点的包围情况
4. 使用奈氏判据：Z = P - N
5. Z=0 则稳定，Z>0 则不稳定（Z 为右半平面闭环极点个数）

## 知识点关联图

```
数学模型（传递函数、框图、信号流图）
    ↓
时域分析（稳定性、动态性能、稳态误差）
    ↓          ↓
根轨迹法    频域分析（Nyquist、Bode）
    ↓          ↓
    └────┬────┘
         ↓
    系统校正（超前/滞后、PID）
         ↓
    离散系统（Z变换、数字控制）
         ↓
    现代控制理论（状态空间、能控能观）
```

- **劳斯判据** 是时域稳定性判据的起点
- **根轨迹** 是参数变化下闭环极点移动的几何轨迹
- **Nyquist 判据** 是频域稳定性判据，与根轨迹互补
- **Bode 图** 是频域分析的工程实用工具
- **校正** 综合运用了前面所有分析手段

## 重点/难点/常考点

- **重点**：传递函数化简（框图等效变换、梅森公式）、劳斯判据、二阶系统时域指标计算
- **难点**：根轨迹绘制（尤其是参量根轨迹）、Nyquist 判据的正确使用、滞后-超前复合校正
- **常考点**：稳态误差计算（型别判断 → 误差系数 → 稳态误差）、Bode 图绘制与稳定裕度计算
