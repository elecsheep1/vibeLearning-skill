# 政治

## 元信息
- learning_mode: exam
- description: 考研政治，涵盖马原、毛中特、史纲、思修、时政等。

## 常见错误类型分类

### 选择题
- **概念偷换**：选项将相似但不同的概念互换。例如"主要矛盾"偷换为"基本矛盾"，"本质"偷换为"特征"。
- **范围扩大/缩小**：将"有利于"说成"决定"，将"一部分"说成"全部"。
- **绝对化表述误判**：选项中含"一定""必然""所有""根本"等词时需警惕，但不是所有绝对词都错（如"实践是检验真理的唯一标准"）。
- **时政时间错位**：将不同时间节点的事件张冠李戴。

### 简答题/论述题
- **原理选取错误**：没有准确识别题目所考察的原理。改进：先圈题目关键词 → 找对应原理。
- **脱离材料**：只默写了原理，完全没有结合材料分析。答题必须"原理 + 材料分析"结合。
- **要点遗漏**：一道题涉及多个原理点时，只写了其中一个。
- **逻辑层次混乱**：答题没有"是什么 → 为什么 → 怎么办"的结构意识。

### 时政
- **时事记忆模糊**：重要会议、文件、时间节点记混。
- **政策表述不准确**：用自己的话替换标准表述导致失分。

## 诊断规则

- 选择题错误集中在某个知识板块 → 该板块基础概念不清，需回归教材
- 大题得分低但字迹工整 → 大概率是"只默不析"：原理堆砌但缺乏材料分析
- 学生说"我都背了但还是不会做" → 背诵停留在字面，缺乏理解性记忆和迁移能力
- 多选题是主要失分点 → 概念辨析能力弱，需针对性做概念对比训练

## 推荐解题步骤模板

### 选择题
1. 读题干，圈出关键词（主体、限定词、核心概念）
2. 判断题干考查的知识板块（马原/毛中特/史纲/思修/时政）
3. 若不熟悉，先排除明显错误的选项
4. 若多选题，逐项验证是否符合题干要求
5. 对标标准表述，警惕偷换概念

### 简答题/论述题
1. **审题**：圈画题目中的核心概念和设问角度
2. **定位原理**：确定考查的是什么原理/知识点
3. **列框架**：按"是什么 → 为什么 → 怎么办"组织答题
4. **动笔写**：
   - 第一段：点明原理（是什么）
   - 第二段：原理展开 + 结合材料分析（为什么）
   - 第三段：总结升华/联系实际（怎么办）
5. **检查**：原理是否准确、材料是否结合、逻辑是否通顺

### 时政题
1. 确认事件所属领域（经济/政治/文化/生态/外交）
2. 回忆该事件的时间线、核心文件和标志性表述
3. 注意区分"首次提出""进一步发展""完善定型"等程度词

## 知识点关联图

```
马克思主义基本原理
    ├── 哲学（唯物论、辩证法、认识论、唯物史观）
    └── 政治经济学（劳动价值论、剩余价值论）

毛泽东思想和中国特色社会主义理论体系
    ├── 毛泽东思想
    ├── 邓小平理论
    ├── 三个代表
    ├── 科学发展观
    └── 习近平新时代中国特色社会主义思想

中国近现代史纲要
    ├── 旧民主主义革命（1840-1919）
    ├── 新民主主义革命（1919-1949）
    ├── 社会主义革命与建设（1949-1978）
    └── 改革开放至今（1978-今）

思想道德与法治
    ├── 人生观、价值观
    ├── 道德规范
    └── 法律基础

形势与政策（时政）
    └── 当年国内外重大事件及政策
```

- **马原**是整个政治理论的方法论基础（尤其是哲学部分）
- **毛中特**是马克思主义中国化的理论成果
- **史纲**是理论的历史背景和事实支撑
- **思修**是理论在个人层面的践行

## 重点/难点/常考点

- **重点**：马原的辩证法三大规律、认识论、真理观；毛中特的新思想"十个明确""十四个坚持"
- **难点**：马原政治经济学（资本有机构成、剩余价值率计算）、史纲的时间线梳理
- **常考点**：矛盾分析法（马原分析题常考）、新思想的核心内容、重要历史节点
