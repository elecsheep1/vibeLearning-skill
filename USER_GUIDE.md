## 一、这是什么？

`vibe-learning` 是一个**科目无关的 AI 辅导引擎**。你只需为每个学科准备一个简单的“科目配置文件”（例如 `高等数学.md`、`规控.md`），AI 就会自动采用对应的教学风格：

- **exam 模式**：考研/考试模式 → 错题诊断、出变式题、输出复习卡片+思维导图
- **professional 模式**：工程实践模式 → 调试代码、项目引导、架构建议
- **research 模式**：学术研究模式 → 论文解读、概念辨析、开放思考题

你正在使用的就是 **exam 模式的高等数学配置**，专门用于考研数学的错题整理和分析。

------

## 二、安装与文件结构

### 1. 存放位置

Skill 文件需要放在 Claude Code 的 skills 目录下：

- **macOS / Linux**：`~/.claude/skills/vibe-learning-skill/`
- **Windows**：`%USERPROFILE%\.claude\skills\vibe-learning-skill\`

### 2. 目录结构（你应该已有）

```text
vibe-learning-skill/
├── SKILL.md                 # 核心引擎（已配置好）
├── subjects/
│   └── 高等数学.md          # 你的高数配置文件
└── templates/               # （可选）存放输出模板
```



> 不需要手动修改 `SKILL.md`，以后只需维护 `subjects/` 下的科目配置文件即可。

## 三、日常使用流程（最重要）

### 场景：整理一道高数错题

1. **拍照/扫描题目**
   用软件（推荐 **Mathpix** 或 **SimpleTex**）将印刷体题目识别成**清晰的图片**（或直接复制识别出的文字）。

2. **打开 Claude Code**
   启动 Claude Code（终端输入 `claude` 或双击应用）。

3. **发送消息**

   - 先告诉 AI 你要学什么：

     ```text
     学习高等数学
     ```

     

   - 上传题目图片，并附上你的错误过程（用语音输入或简短打字）：

     ```text
     这道积分题我写的是：∫ x e^x dx = ½ x² e^x + C  
     正确答案是 e^x (x-1) + C  
     整理错题
     ```

     

4. **AI 输出**
   AI 会直接给你：

   - 一张 **错题卡片**（知识点、错误类型、根因、正确思路、易错点、解题套路、变式题）
   - 一张 **思维导图**（Mermaid 格式，可以复制到支持 Mermaid 的笔记软件中查看）
   - 询问是否保存到错题本

5. **保存结果**
   你可以把卡片内容复制到自己的笔记文件（例如 `错题集/2025-01-20-分部积分.md`），方便以后复习。

### 场景：遇到配置中没有的新错误

当 AI 诊断后，如果它发现这种错误在 `高等数学.md` 中尚未记录，会主动询问：

> “我注意到本次错误类型在当前科目配置中尚未收录。是否需要我生成一段配置内容，添加到 `subjects/高等数学.md` 中？”

- 你只需回复 **“是”** 或 **“好”**

- AI 会输出一段完整的配置片段，例如：

  ```markdown
  ## 可追加到”积分”章节的内容
  - **分部积分选u错误**：...  
  ### 新增诊断规则
  - 用户使用分部积分后积分变复杂 → 选u顺序错误
  ```

  

- 你复制这段内容，打开 `subjects/高等数学.md`，粘贴到对应章节末尾，保存即可。

> 整个过程不到 20 秒，以后 AI 就能自动识别这类错误了。

### 场景：想学习其他科目（如规控）

1. 创建 `subjects/规控.md`（可参考你已有高数配置的格式，但 `learning_mode` 改为 `professional`）
2. 在 Claude Code 中说：`学习规控`
3. 之后就可以粘贴代码、问调试问题，AI 会切换为工程实践模式。

### 场景：临时切换学习模式

如果你想用 exam 模式分析规控中的公式推导错误（而非 professional 的调试模式），可以说：

> “虽然现在学习规控，但请用 exam 模式帮我整理其中的公式推导错误”

AI 会在本次会话中临时覆盖模式。

## 四、常见问题

### Q1：AI 没有自动输出卡片，而是长篇对话？

**A**：你可能没有明确说“整理错题”。请确保在提供错题后加上指令：`整理错题` 或 `diagnose`。

### Q2：上传的图片识别不准怎么办？

**A**：Claude 对**清晰印刷体**识别率很高。如果识别有误，你可以：

- 用 Mathpix 重新生成更清晰的图片
- 或者直接把识别出的文字粘贴到对话中

### Q3：如何让 AI 不保存错题本就提问？

**A**：当 AI 询问“是否保存到错题本”时，回复“不用”即可，不会影响分析结果。

### Q4：科目配置文件中的“诊断规则”怎么写？

**A**：最简单的方法：让 AI 帮你生成。每次遇到新错误时，同意 AI 的“是否添加配置”建议，它自动生成的片段就是标准格式。你也可以直接仿照已有的规则写。

### Q5：我想批量处理多道错题怎么办？

**A**：目前 v2.2 还不支持自动批量。你可以：

- 一道一道发，每次说”整理错题”
- 或者把所有错题的文字（题目+错误过程）放在一个文本文件中，用 `读取文件` 命令一次性发给 AI，然后说”请逐道整理”。

批量功能将在 v2.3 中支持。

### Q6：如何评估这个 Skill 对我有没有帮助？

**A**：Skill 内置了评估标准。你可以每两周抽样几道错题，记录：

- AI 输出的卡片是否完整
- 错误类型分类是否正确
- 你根据诊断能否自己改对变式题
- 每次用时多少秒

如果你觉得某个方面不满意（例如根因分析太浅），可以补充科目配置中的“诊断规则”来改进。

## 五、进阶技巧

### 1. 语音输入错误过程

在手机或电脑上用语音输入描述错误步骤（“我第一步直接乘积积分了”），比打字快得多。

### 2. 利用模板自定义卡片格式

如果你希望卡片顺序不同或增加字段，可以修改 `templates/mistake-analysis.md`（如果没有就创建一个）。Skill 会优先使用模板中的格式。

### 3. 将错题卡片同步到 Obsidian / Notion

AI 输出的卡片是 Markdown 格式，你可以直接复制粘贴到任何笔记软件中。如果使用 Obsidian，还可以安装 Mermaid 插件来渲染思维导图。

### 4. 定期维护配置文件

每个月花 10 分钟，把 AI 建议添加的配置片段整理到对应的分类章节，删除重复项。你的配置文件会越来越智能。
