# vibeLearning-skill
> 一个科目无关的通用 AI 学习辅导引擎。

最近在准备考研复习，利用AI顺手做了一个帮助自己整理错题和总结知识的 Skill。可能还不太成熟。如果你也在用 Claude Code，欢迎试用和反馈～

## ✨ 主要功能

- **科目无关，动态加载**：只需一个配置文件，就能让 AI 掌握任意学科的教学风格。
- **三种学习模式 + Professional 子模式**：
  - `exam` – 考试模式：错题诊断、输出复习卡片 + 思维导图、生成变式题
  - `professional` – 工程模式，含两个子模式：
    - `project` – 项目驱动：代码调试、项目引导、架构建议
    - `qa` – 问答驱动：用户来问 AI 来答，渐进式构建知识体系
  - `research` – 研究模式：概念辨析、论文解读、开放思考题
- **错题自动分析**：上传印刷体题目图片，AI 会定位错误步骤、分类错误类型、分析根因并给出改进建议。
- **配置自进化**：遇到新错误时，AI 会主动询问是否生成配置片段，你只需复制粘贴就能让"错题库"越来越聪明。
- **学习笔记实时同步**（qa 模式）：每轮问答结束后自动写入学习笔记，按主题分文件，时间序累积，形成完整的学习日志。
- **XMind 思维导图**：知识树自动生成 `.xmind` 格式，可在 XMind 中打开浏览、折叠、编辑和标记学习进度。
- **模块化学习目录**：笔记和知识树按科目自动组织为 `<科目>/笔记/` + `<科目>/知识树/` 的目录结构，支持 LaTeX 数学公式渲染。
- **支持图片输入**（推荐）和文本输入。

## 📦 安装

1. 将本仓库克隆到 Claude Code 的 skills 目录：

```
git clone https://github.com/yourname/vibe-learning-skill.git ~/.claude/skills/vibe-learning-skill
```

2. 重启 Claude Code（或输入 `/skills` 刷新列表）。
3. 为你想学习的科目创建配置文件（参考 `subjects/高等数学.md` 示例）。

## 🚀 快速使用

详细的使用教程请参考：[USER_GUIDE](./USER_GUIDE.md)

### 学习新知识时，AI 会自动生成模块化结构：

```
<工作区>/
├── <主学习规划文档>.md
├── <科目名>/                  ← 自动创建
│   ├── 笔记/                  ← 学习笔记按主题存放
│   │   ├── <主题1>-学习笔记.md
│   │   └── <主题2>-学习笔记.md
│   └── 知识树/                ← 知识树大纲 + XMind 导图
│       ├── <科目名>知识树.md       ← 文本版，可版本控制
│       └── <科目名>知识树.xmind    ← XMind 可视化
```

所有文件之间用相对路径交叉引用，知识树用 ⬜ → 🟡 → ✅ 追踪每个节点的学习进度。

## ⚙️ 配置文件格式（极简）

```
# 科目名称

## 元信息
- learning_mode: exam | professional | research | general
- description: 一句话描述

## 常见错误类型分类
- 错误名称：描述、典型表现、纠正方法

## 诊断规则
- 用户出现 X → 判定为 Y 错误

## 知识点关联图
- 知识点A → 前置：[B] → 后续：[C]
```

更完整的例子请参考 `subjects/高等数学.md`。

## 🆕 最近更新 (v2.4)

- **Professional 子模式**：进入工程类科目时，AI 会询问采用 `project`（项目驱动）还是 `qa`（问答驱动）学习方式
- **问答式学习（qa）**：用户来问、AI 来答，结构化回答含知识关联 + 工程视角 + 延伸问题，知识树自动追踪进度，**每轮 Q&A 自动写入学习笔记**
- **子模式动态切换**：学习中可随时说"切换 project"或"切换 qa"
- **科目配置新增 `sub_mode`**：可预设默认子模式，无需每次手动选择

### 历史更新 (v2.3)

- **XMind 思维导图**：知识树输出格式从 Mermaid 迁移到 `.xmind`，支持在 XMind 中直接打开编辑
- **模块化目录结构**：学习内容自动按 `<科目>/笔记/` + `<科目>/知识树/` 组织
- **LaTeX 公式支持**：笔记中数学公式统一使用标准 LaTeX 语法（`$\mathcal{S}$`, `$\mathbb{E}$`, `$\sum$` 等）
- **新增科目配置**：强化学习（`professional` 模式），含完整的 MDP/Bellman/算法分类等知识点关联
